# extension-quick-toggle

SillyTavern 扩展快捷开关，在设置菜单底部添加一键切换按钮，快速启用/禁用指定扩展。

## 功能

- 一键开关 **聊天记录管理器** (vectors-enhanced)
- 一键开关 **小白X** (LittleWhiteBox)
- 点击即切换，**即时生效，无需刷新页面**
- 按钮状态实时显示当前启用/禁用状态

## 安装

在酒馆中：**扩展 → 安装扩展 → 输入仓库地址**

```
https://github.com/relax-sketch/extension-quick-toggle
```

## 使用

1. 打开酒馆设置菜单（齿轮图标）
2. 滚动到底部，在横线下方可见两个开关按钮
3. 绿色图标 = 已启用，灰色图标 = 已关闭
4. 点击按钮即时切换，无需刷新页面

## 卸载

在酒馆中：**扩展 → 扩展管理 → 找到"扩展快捷开关" → 删除**

## 原理

直接操作扩展内部的**主开关**，而非扩展级别的启用/禁用：

- **聊天记录管理器** → `master_enabled` 主开关（等同界面上的"启用聊天记录超级管理器"）
- **小白X** → `enabled` 主开关（等同界面上的"关闭小白X"）

切换时即时生效，无需刷新页面。

## 兼容性

- SillyTavern 1.15.0+
- 需要已安装 vectors-enhanced 和/或 LittleWhiteBox 扩展
- 如果某个扩展未安装，对应按钮不会显示
