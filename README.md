# extension-quick-toggle

SillyTavern 扩展快捷开关，在设置菜单底部添加一键切换按钮。

## 功能

| 扩展 | 切换方式 | 说明 |
|------|---------|------|
| **聊天记录管理器** (vectors-enhanced) | 软切换 | 操作内部 `master_enabled` 开关，即时生效 |
| **小白X** (LittleWhiteBox) | 软切换 | 操作内部 `enabled` 开关，即时生效 |
| **WestWorld** | 硬切换 | 启用/禁用扩展，自动刷新页面 |

## 安装

在酒馆中：**扩展 → 安装扩展 → 输入仓库地址**

```
https://github.com/relax-sketch/extension-quick-toggle
```

## 使用

1. 打开酒馆设置菜单（齿轮图标）
2. 滚动到底部，在横线下方可见开关按钮
3. 绿色/蓝色图标 = 已启用，灰色图标 = 已关闭
4. 点击切换 — 软切换即时生效，硬切换自动刷新

## 卸载

在酒馆中：**扩展 → 扩展管理 → 找到"扩展快捷开关" → 删除**

## 兼容性

- SillyTavern 1.15.0+
- 如果某个扩展未安装，对应按钮不会显示
